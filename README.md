## 🏁 Getting Started <a name = "getting_started"></a>


<hr>

### Prerequisites

- [Node.js](https://nodejs.org/): ^12.0.0
- [NPM](https://npmjs.org/) or any other Node.js package manager.

<hr>

### Installing

Install packages npm:


```
npm install
```

Then start the dev server with this command:

```
npm run dev
```
